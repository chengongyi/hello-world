------------------------------------
bower 前端包管理器的使用

#注意：！！！！！！！
当前的文件使用目录，不能有中文字符


#安装
cnpm install -g bower

#查看包信息
bower info 

#安装vue文件，并且指定文件版本号
bower install vue#1.0.28
bower install vue@1.0.28


注意点1：
在mac下面 执行cnpm install -g vue-cli语句的时候，有可能会报错
错误信息为： 
Error: EACCES: permission denied, unlink '/usr/local/lib/node_modules/vue-cli/.babelrc'
